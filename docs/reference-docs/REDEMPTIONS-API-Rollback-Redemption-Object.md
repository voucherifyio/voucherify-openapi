## Rollback Redemption Object
<p>This is an object representing an attempted or successful voucher or promotion tier rollback.</p>

Any of:

[Discount Voucher](#discount-voucher), [Loyalty Card Voucher](#loyalty-card-voucher), [Gift Card](#gift-card), [Promotion Tier](#promotion-tier)
