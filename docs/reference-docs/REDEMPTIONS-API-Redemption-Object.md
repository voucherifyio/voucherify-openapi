## Redemption Object
<p>This is an object representing a redemption.</p><p>Redemption is the key operation in the voucher and promotion tier lifecycle. A customer can redeem a voucher or promotion tier once or multiple times depending on selected limit (<code>quantity</code>). Each redemption is recorded in voucher/promotion's history (<code>redemption_entries</code>). There is also an option to cancel a redemption. We call such an operation a <!-- [redemption rollback](OpenAPI.json/paths/~1redemptions~1{redemptionId}~1rollback/post) --><a href="ref:rollback-redemption">redemption rollback</a>.</p>

Any of:

[Discount Voucher](#discount-voucher), [Loyalty Card Voucher](#loyalty-card-voucher), [Gift Card](#gift-card), [Promotion Tier](#promotion-tier), [Stacked Redemption](#stacked-redemption), [Failed Voucher Redemption](#failed-voucher-redemption), [Failed Promotion Tier Redemption](#failed-promotion-tier-redemption), [Failed Redemption](#failed-redemption)
