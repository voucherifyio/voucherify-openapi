## Concept

The Voucherify and Braze integration separates two distinct responsibilities:

- **Voucherify (Optimization Layer)**: Handles incentive logic, eligibility validation, real-time decisioning, guardrails, usage limits, and cross-channel enforcement.
- **Braze (Engagement Layer)**: Handles segmentation, journey orchestration, delivery, A/B testing, and messaging.

This separation lets you run experiments in Braze while Voucherify enforces margin protection limits across all channels.

With this integration, you can:
- Trigger messages using Action Based Delivery
- Synchronize Braze audience IDs to customer metadata for targeted incentive configurations
- Push validated incentives to Braze using:
  - [Braze's Connected Content](https://www.braze.com/docs/user_guide/personalization_and_dynamic_content/connected_content/)
  - [Braze's Custom Attributes](https://www.braze.com/docs/user_guide/data_and_analytics/custom_data/custom_attributes/)
  - [Promo Codes Snippet](https://www.braze.com/docs/user_guide/personalization_and_dynamic_content/promotion_codes/)

<Note>

    The integration with Braze consumes **[API limits](/guides/limits#api-limits)** for Connected Content and **[webhook limits](/guides/limits#webhooks)** for sending codes to Braze with custom attributes and synchronizing audiences.

</Note>

## Integration

Voucherify and Braze communicate both ways. You need to connect both platforms.

### Connect Voucherify with Braze

To connect Voucherify with Braze:
1. Copy the REST API key from your Braze account (requires at least the `users.track` permission).
2. In Voucherify, go to **Home** > **Integrations** and find Braze to **Connect**.
3. Paste the copied REST API key from Braze, select the API endpoint, enable **Send events to Braze**, and select **Connect**.

### Connect Braze with Voucherify

Braze may require your Voucherify API keys in various settings like Canvas or Connected Content.

To get your API integration keys from Voucherify:
1. In **Home**, go to **Project Settings** and scroll down to **Integration keys**.
2. Click the **+** button to open the key generation panel.
3. **Name** your key, assign the **Role** (predefined or custom), and select **Braze** from the list of available integrations to **Create integration API**.
4. Save your application ID and secret key securely. You will not be able to view the key again.
