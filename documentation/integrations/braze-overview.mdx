## Concept

This integration connects your Incentive Optimization Engine (Voucherify) with your Engagement Layer (Braze) to enable continuous incentive experimentation.

**Voucherify handles the Optimization Layer**:
- Incentive logic and configuration
- Eligibility validation
- Real-time decisioning
- Guardrails (margin protection limits, redemption caps, expiration logic)
- Cross-channel enforcement

**Braze handles the Engagement Layer**:
- Audience segmentation
- Journey orchestration
- Delivery and testing
- Messaging

You can trigger validated incentives to Braze through:
- [Braze's Connected Content](https://www.braze.com/docs/user_guide/personalization_and_dynamic_content/connected_content/)
- [Braze's Custom Attributes](https://www.braze.com/docs/user_guide/data_and_analytics/custom_data/custom_attributes/)
- [Promo Codes Snippet](https://www.braze.com/docs/user_guide/personalization_and_dynamic_content/promotion_codes/)

You can also synchronize Braze audience IDs to customer metadata in Voucherify to run experiments targeting specific customer groups segmented in Braze.

<Note>

This integration consumes **[API limits](/guides/limits#api-limits)** for Connected Content and **[webhook limits](/guides/limits#webhooks)** for triggering incentives and synchronizing audiences.

</Note>

## Integration

Voucherify and Braze communicate bidirectionally. Configure both connections to enable continuous incentive experimentation.

### Connect Voucherify with Braze

1. Copy the REST API key from your Braze account (requires at least `users.track` permission).
2. In Voucherify, go to **Home** > **Integrations** and find Braze to **Connect**.
3. Paste the REST API key, select API endpoint, and enable **Send events to Braze** to **Connect**.

### Connect Braze with Voucherify

Braze requires Voucherify API keys for Canvas workflows and Connected Content.

1. In Voucherify **Home**, go to **Project Settings** and scroll to **Integration keys**.
2. Select **+** to open the key generation panel.
3. Name your key, assign a **Role**, and select **Braze** from the integrations list to **Create integration API**.
4. Store your application ID and secret key securely. You cannot view the key again after creation.
