The Voucherify-Braze integration lets you move data between both platforms. You can stream promotion events to Braze, sync customer segments, and deliver Voucherify incentives directly in Braze messages.

You can push Voucherify incentives to Braze with:
- [Connected Content](/integrations/braze-connected-content-overview) to fetch codes at send time
- [Custom attributes](/integrations/braze-custom-attributes) to store codes on Braze user profiles
- [Promotion code lists](/integrations/braze-promotion-codes) to import codes in bulk

<Note>

    The integration with Braze consumes **[API limits](/guides/limits#api-limits)** for Connected Content and **[webhook limits](/guides/limits#webhooks)** for sending codes to Braze with custom attributes, as well as synchronizing the audience in Braze.

</Note>

## What you need to connect with Braze

Voucherify and Braze communicate both ways, so you need to connect both platforms.

### Connect Voucherify with Braze

To connect Voucherify with Braze:
1. Copy the REST API key from your Braze account (it should at least have the `users.track` permission).
2. In Voucherify, go to **Home** > **Integrations** and find Braze to **Connect**.
3. Paste the copied REST API key from Braze, select API endpoint, tick Send events to Braze to **Connect** your platforms.

### Connect Braze with Voucherify

Braze may require your Voucherify API keys in various settings like Canvas or Connected Content.

To get your API integration keys from Voucherify:
1. In **Home**, go to **Project Settings** and scroll down to **Integration keys**.
2. Click the **+** button to open the key generation panel.
3. **Name** your key, assign the **Role** (predefined or a custom one), and select **Braze** from the list of available integrations to **Create integration API**.
4. Save your application ID and secret key in a safe place. You will not be able to view the key again.

## Rate limiter

When configuring campaigns in Braze, [limit the number of messages](https://www.braze.com/docs/user_guide/engagement_tools/campaigns/building_campaigns/rate-limiting/#delivery-speed-rate-limiting) sent per minute. This protects both Braze and Voucherify APIs from excessive traffic. Set the sending rate based on your Voucherify [API rate limits](/guides/limits#api-limits) and account for other API call needs.
