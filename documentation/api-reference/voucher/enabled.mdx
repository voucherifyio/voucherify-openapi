The `EVENTS.VOUCHER.ENABLED` event indicates that a voucher has been successfully enabled for use. It indicates that the voucher is now active and it can be redeemed.

This event is used in a webhook configured in Project settings in Voucherify dashboard.

<note>This page documents only the event. If you need more details about the webhook payload data that includes this event, go to [Webhook v2024-01-01](/api-reference/introduction-to-webhooks "Introduction to webhooks v2024-01-01") page.</note>
