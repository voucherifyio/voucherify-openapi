The `EVENTS.CUSTOMER.LOYALTY.TIER.LEFT` event indicates that a customer has exited a loyalty campaign tier structure or a specific tier.

This event is used in a webhook configured in Distributions in Voucherify dashboard.

<note>This page documents only the event. If you need more details about the webhook payload data that includes this event, go to [Webhook v2024-01-01](/api-reference/introduction-to-webhooks "Introduction to webhooks v2024-01-01") page.</note>
