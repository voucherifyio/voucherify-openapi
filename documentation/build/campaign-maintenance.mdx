Use the campaign dashboard to manage your campaigns after launch. This guide applies to all campaign types: discount coupons, cart promotions, gift cards, loyalty programs, and referral programs.

## Prerequisites

- An active Voucherify project with at least one campaign
- Dashboard access with campaign management permissions

## Access the campaign dashboard

Go to **Campaign hub** > **Campaigns** and select a campaign name to open its dashboard.

From the dashboard, you can:

- View real-time performance metrics
- Manage individual codes (vouchers, loyalty cards, referral codes, gift cards)
- Edit campaign settings
- Set up distributions
- Export campaign data

<Note>
Managing individual codes refers to enabling, disabling, deleting, and exporting campaign assets. While loyalty cards are not always labeled as "codes" in the UI, they are managed in a similar way from the dashboard.
</Note>

## Campaign lifecycle

Campaigns move through these states:

| State | Description |
|-------|-------------|
| **Draft** | Campaign is being set up. No codes are generated yet. |
| **Active** | Campaign is live. Codes can be redeemed. |
| **Disabled** | Campaign is paused. No redemptions allowed. |
| **Deleted** | Campaign is removed (to bin or permanently). |

<Tip>
Use **Disable** when you need to pause a campaign temporarily. Use **Delete** only when the campaign is no longer needed.
</Tip>

## Edit an active campaign

You can modify certain campaign settings after launch without affecting issued codes.

### What you can change

| Setting | Impact on existing codes |
|---------|-------------------------|
| Campaign name | No impact |
| Description and metadata | No impact |
| Start and end dates | Affects all codes |
| Validation rules | Affects all future redemptions |
| Category assignment | No impact |

</Step>
<Step title="Edit settings">

## Enable or disable a campaign

### When to disable vs delete

| I want to... | Action |
|--------------|--------|
| Pause redemptions temporarily | **Disable** |
| Stop the campaign but keep history | **Move to bin** |
| Remove the campaign and reuse the name/codes | **Delete permanently** |

### Disable a campaign

A disabled campaign prevents all redemptions and reward triggering. Vouchers, gift cards, referral rewards, and loyalty rewards from a disabled campaign cannot be redeemed or issued until you re-enable it.

Disabling a campaign does **not** remove already issued vouchers, gift card balances, loyalty points, or referral relationships—it only stops further earning, redemption, or reward fulfillment.

<Steps>
<Step title="Open the campaign">
Go to **Campaign hub** > **Campaigns** and select the campaign name.
</Step>
<Step title="Disable the campaign">
Select **Disable** to stop redemptions.
</Step>
</Steps>

### Re-enable a campaign

<Steps>
<Step title="Open the disabled campaign">
Go to **Campaign hub** > **Campaigns** and select the disabled campaign.
</Step>
<Step title="Enable the campaign">
Select **Enable** to allow redemptions again.
</Step>
</Steps>

<Note>
Re-enabling a campaign restores full functionality. All previously issued codes become redeemable again, subject to their individual validity settings.
</Note>

### What happens when you disable a campaign

| Asset type | Effect |
|------------|--------|
| Vouchers | Cannot be redeemed |
| Gift cards | Balance is frozen; cannot add or redeem |
| Loyalty cards | Cannot earn or redeem points |
| Referral codes | Referrals are not tracked; rewards not issued |

## Clone a campaign

Clone an existing campaign to create a new one with similar settings.

<Steps>
<Step title="Select the campaign">
Go to **Campaign hub** > **Campaigns** and select the campaign you want to clone.
</Step>
Select **Clone** from the action menu.
</Step>
<Step title="Customize">
Modify the cloned campaign settings as needed.
</Step>
</Steps>

<Note>
Referral programs cannot be cloned.
</Note>

## Delete a campaign

<Steps>
<Step title="Open the campaign">
Go to **Campaign hub** > **Campaigns** and select the campaign name.
</Step>
<Step title="Delete">
Select the delete icon, or select the three dots (⋮) > **Delete** from the campaign list.
</Step>
<Step title="Choose deletion method">
Choose a deletion method:
- **Move to bin**: Prevents reusing the same campaign name and codes in this project. You can restore or permanently delete items from the bin later.
- **Delete permanently**: Allows you to reuse the campaign name and codes in future campaigns.
</Step>
</Steps>

<Warning>
Permanent deletion cannot be undone. Choose this option only if you want to reuse the same campaign name or codes.
</Warning>

### What happens when you delete a campaign

| Deletion type | Issued vouchers | Gift card balances | Loyalty points | Referral relationships |
|---------------|-----------------|-------------------|----------------|----------------------|
| Move to bin | Preserved, not redeemable | Preserved, frozen | Preserved, frozen | Preserved, inactive |
| Delete permanently | Removed | Removed | Removed | Removed |

For more information about managing deleted items, see [Bin](/prepare/bin).

## Export campaign data

Export campaign data for external analysis or reporting. For detailed reporting options, see [Campaign reporting](/guides/campaign-reporting).

### Export vouchers or codes

<Steps>
<Step title="Open the campaign dashboard">
Go to the campaign dashboard.
</Step>
<Step title="Select the tab">
Open the relevant tab (**Vouchers**, **Loyalty Cards**, **Referral codes**).
</Step>
<Step title="Export">
Select the **Export** tool and choose your export format and destination.
</Step>
</Steps>

### Export transactions

For gift cards and loyalty programs:

<Steps>
<Step title="Open transactions">
Go to the campaign dashboard and open the **Transactions** tab.
</Step>
<Step title="Export">
Select the **Export** tool.
</Step>
</Steps>

### Export points expiration

For loyalty programs:

<Steps>
<Step title="Open points expiration">
Go to the campaign dashboard and open the **Points Expiration** tab.
</Step>
<Step title="Export">
Select the **Export** tool.
</Step>
</Steps>

## Organize with categories

Use categories to group and filter vouchers quickly.

- Create categories when creating a campaign or in the voucher detail view.
- Manage all categories in **Campaigns** > **Categories**.
- Categories have a hierarchy number. Lower numbers indicate higher importance.

For more information, see [Manage categories](/orchestrate/manage-categories).

## Automate with webhooks

Set up webhooks to receive notifications when campaign events occur:

| Event | Description |
|-------|-------------|
| `campaign.enabled` | Campaign was enabled |
| `campaign.disabled` | Campaign was disabled |
| `campaign.deleted` | Campaign was deleted |
| `campaign.updated` | Campaign settings were changed |

For webhook setup instructions, see [Project settings webhooks](/api-reference/project-settings-webhooks).
