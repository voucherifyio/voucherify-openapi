title: "Create a Friday happy hour discount"
description: "Step-by-step guide to set up a 10% discount coupon valid only on Fridays between 10AM-3PM"
---

This guide shows you how to create a discount coupon that works only during specific hours on specific days.

**Example scenario**: 10% off on Fridays from 10:00 AM to 3:00 PM.

## Prerequisites

- Access to the Voucherify Dashboard
- Permission to create campaigns

## Create the campaign

<Steps>

<Step title="Start a new campaign">

Go to **Campaign hub** > **Campaigns** > **Create campaign** and select **Discount coupons**.

</Step>

<Step title="Configure campaign details">

1. Enter a **Campaign name** (e.g., "Friday Happy Hour 10% Off").
2. Choose **Generic code** for a single code or **Bulk codes** for multiple unique codes.

</Step>

<Step title="Set time limitations">

In the **Time frame** section:

1. Set **Starts on** to your desired start date.
2. Set **Expires** to define when the campaign ends (or select **Never**).
3. Click **Add limitation** and select **Add valid hours per day**.
4. Configure the following:
   - Select **Friday** as the valid day.
   - Set **Start time** to `10:00`.
   - Set **End time** to `15:00`.

<Note>
All times follow the time zone defined in your **Project settings**.
</Note>

</Step>

<Step title="Configure the discount">

In the **Discount value** section:

1. Select **Order discount**.
2. Choose **Percent** discount type.
3. Enter `10` as the percentage value.

</Step>

<Step title="Save and launch">

Review the **Summary** and click **Save** to create the campaign.

</Step>

</Steps>

## Result

Customers can redeem the coupon only on Fridays between 10:00 AM and 3:00 PM. Redemption attempts outside this window are blocked.
