Use Voucherify tools to find, manage, and update your existing vouchers and campaigns.

## Find vouchers

Use the Vouchers view to browse and search all voucher types in one place.

Go to **Campaign Hub** > **Campaigns** > **Vouchers** to see all your codes, including:

- discount coupons
- gift cards
- referral codes
- loyalty cards

### Use categories

Use categories to organize vouchers and filter them faster.

You can create a category when you create a campaign or add it later in the voucher details. To manage categories, go to **Campaigns** > **Categories**.

Categories can also have a hierarchy. The lower the number, the higher the category priority.

### Search by code

Use the search bar to quickly find a voucher by its code.

The search box is available on the left below the currently selected project.

### Filter vouchers

Use filters to narrow down the list of vouchers based on selected criteria.

In the **Vouchers** view, select **Add filter** to build your search.

<Accordion title="Available filters">

You can filter vouchers by:

- voucher code
- voucher type
- redeemed quantity
- campaign
- campaign status
- category
- start date
- expiration date
- creation date
- update date
- voucher status, for example active or expired
- custom attributes added as metadata

</Accordion>

You can use many filters at the same time.

For example, you can find:

- expired gift cards
- active loyalty cards
- vouchers created in a selected time range

<Accordion title="Bulk input and filter editing">

You can also use **Bulk input** with the **in** condition in metadata filters.

Bulk input supports:

- colon `:`
- semicolon `;`
- comma `,`
- new line

You can paste up to 200 values.

Bulk input works with string and number properties. For dictionary metadata, only valid values are accepted.

You can edit any filter chip by selecting it. You can change:

- property
- condition
- expected value

Changes are saved automatically, and the results refresh right away.

</Accordion>

### Find generic codes

Use filters to find generic codes that are not linked to a campaign.

To list generic codes, set the filter **Campaign is unknown**.

For newly created generic codes, you may need to pass a campaign name when searching.

## Manage vouchers

Use voucher actions to control availability or remove codes when needed.

### Enable or disable a voucher

Enable or disable vouchers directly from the campaign view.

1. Open the campaign details.
2. Go to the **Vouchers** tab.
3. Use the toggle next to a voucher.

### Delete a voucher

Delete vouchers when they are no longer needed.

1. Select the three dots **⁝** next to the voucher.
2. Select **Delete**.
3. Choose one of the following options:

- **Move to bin** – soft delete. You cannot reuse the code yet.
- **Delete permanently** – full delete. You can reuse the code later.

## Manage campaigns

Use campaign actions to control how voucher groups behave.

### Search campaigns

Use filters to find campaigns based on selected criteria.

Go to the **Campaigns** view and select **Add filters**.

<Accordion title="Available campaign filters">

- name
- type
- bulk or generic
- start date
- expiration date
- creation date
- area and store assignment
- metadata

</Accordion>

### Enable or disable a campaign

Enable or disable campaigns to control voucher usage.

When a campaign is disabled, its vouchers cannot be redeemed.

### Clone a campaign

Clone campaigns to reuse configuration and save time.

1. Go to the **Campaigns** view.
2. Open a campaign.
3. Duplicate it.

Referral programs cannot be cloned.

### Delete a campaign

Delete campaigns when they are no longer needed.

You can delete a campaign from the details view or the campaign list.

Choose:

- **Move to bin** – soft delete. You cannot reuse the campaign name or codes until it is permanently deleted from the bin.
- **Delete permanently** – full delete. You can reuse the campaign name and codes.
