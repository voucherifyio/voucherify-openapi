Team settings control your whole Voucherify account. Only the **account owner** can open and manage Team settings.

To open Team settings, open the profile menu in the bottom-left corner of the dashboard and select **Team settings**.

## Subscription

The Subscription section lets you manage your plan, billing details, and usage.

- Create, update, or cancel a subscription with **Create subscription**
- Update your payment method
- Edit billing details in the Billing details section
- Review usage and limits

Your billing cycle starts on the day you confirm your subscription. Usage limits reset at the start of each cycle.

If you exceed your usage limits, API access is blocked until the next cycle starts or you upgrade your plan.

### API usage overview

Team settings show usage across projects (API, messages, and integration webhooks). Use the project selector (the down arrow) to select a project and view its usage.

Set notifications for usage (API, messages, and integration webhooks). Under the charts, select **Set notifications** to define:
- the threshold that triggers a notification (percentage of your limit)
- notification channel (email or in-app notification in the dashboard)
- recipients

For project-level usage details, go to **Project settings > API usage** in a selected project.

## Security

The Security section lets you enforce account-level security rules and check if all users are enrolled.

- Enforce two-factor authentication (2FA) for your team

Voucherify supports:
- SMS codes
- Google Authenticator
- backup codes

## Invoices

The Invoices section lets you review and download invoices.

Voucherify issues invoices based on your subscription (monthly or annual). Find them in the Invoices tab and download them.

## Team

The Team section lets you manage account users.

- Use filters to find team members
- Invite a new user with **Invite member**
- Edit or remove a user using the three-dots (actions) menu

Removing a user is a two-step process:
1. Delete the user
2. Use the actions menu again to delete them permanently

When you invite a user, select the projects they can access and assign a role for each project.

## Roles

The Roles section lets you review role permissions and manage custom roles.

### Owner flag

Owner is a flag on a user account that gives access to Team settings.

Ownership can be granted only by Voucherify support on request from a current owner.

It is recommended to have more than one owner so there is always at least one person with access to Team settings.

### Built-in roles

When assigning a role to a user in a project, select from built-in roles:
- **Admin**: full access to projects and Project settings (cannot access Team settings)
- **User**: can read and change data in assigned projects
- **Viewer**: can read and export data, but cannot change it
- **Merchant**: can validate and redeem incentives
- **Restricted user**: required when using areas and stores; access is limited to assigned areas or stores set in Project settings

### Custom roles (enterprise)

On enterprise plans, you can create custom roles. Custom roles are defined for the whole account and assigned to users per project.

To manage roles, go to **Team settings > Roles**:
- add a new role by duplicating an existing role
- edit custom roles (built-in roles cannot be edited)

## Projects

The Projects section lets you view and filter all projects in your account.

To modify project configuration, go to **Project settings** for a selected project.

Data is not shared between projects. Each project has separate customers, campaigns, and configuration.

- Create a new project with **Create project** (if your subscription limits allow it)
- Switch between projects using the actions menu to start working in another project
- Delete a project using the actions menu (you must be in a different project to delete it; otherwise, the option is disabled)

### Create a project

When you create a project, set:
- name
- description (optional)
- region (Europe, United States, or Asia (Singapore))
- time zone
- currency
- default international dialing prefix (optional)

You also set voucher code case sensitivity as a permanent setting:
- case sensitive: `cOdE1` and `CODE1` are different codes
- case insensitive: `cOdE1` and `CODE1` are the same code

<Warning>
You cannot change voucher code case sensitivity after you create the project.
</Warning>

During project creation, assign users and define their roles.

## Legal & compliance

The Legal & compliance section lets you review legal documents and add compliance contact details.

Voucherify is a GDPR-compliant data processor.

- Review policies, subscription agreements, and data processing agreements
- Review and accept updates to policies
- Add contact details such as:
  - EU representative
  - data protection officer
  - security contact person
- Access compliance documents available in the dashboard

## Delete people data

The Delete people data section lets you permanently remove customer history for GDPR requests.

Search for a customer using **source_id**, then delete and confirm.

## Management API (enterprise)

The Management API section lets you review enterprise-level administrative API scope and usage.

The Management API is available on enterprise plans.

It provides endpoints to manage:
- projects
- users
- schemas
- integration webhooks
- branding

You can review Management API usage in the Subscription tab.
