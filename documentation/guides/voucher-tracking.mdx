This article focuses on voucher-level activity such as redemptions, transactions, and exports. For campaign performance and analytics, see [Campaign tracking]().

## Track campaign activity

Use built-in views to understand what is happening in your campaigns.

### Activity

Use the Activity tab to review all campaign events.

Events are listed in chronological order, with the newest first.

You can filter events by:

- name
- category
- occurrence date

### Campaign Calendar

Use the Campaign Calendar to view campaign timelines.

Go to **Marketer Hub** > **Campaign Calendar** to see:

- active campaigns
- scheduled campaigns
- start and end dates

Select a campaign bar to see more details.

## Track gift card and loyalty card transactions

Use the Transactions tab to monitor balance and point changes.

### Transactions overview

Gift card and loyalty campaigns include a **Transactions** tab in the campaign view. The Transactions tab shows all operations on gift cards and loyalty cards.

You can:

- review history
- check balances
- export data

### Gift card transactions

Gift card transactions track balance changes on cards.

They include:

- redemption
- refund
- addition
- removal

### Loyalty card transactions

Loyalty transactions track changes in points.

Loyalty points can be:

- redeemed
- adjusted
- transferred

### Campaign transactions

The campaign **Transactions** tab shows all transactions in one place.

In the table, each transaction shows:

- **Code** – the voucher code linked to the transaction
- **Type** – the type of transaction
- **Source ID** – custom transaction identifier
- **Reason** – optional description
- **Amount** – funds or points added or removed
- **Balance** – card balance after the transaction
- **Created at** – time the transaction was created

Select the expand icon at the end of a row to view full transaction details in JSON format, including related objects such as orders, redemptions, and balance changes.

Use this view to review transaction history and inspect technical details when needed.

## Export tracking data

Use exports to download voucher and transaction data for reporting.

### Export transactions

Export transactions directly from the **Transactions** tab.

Select **Export**, then choose which fields to include in the CSV file.

<Accordion title="Available export fields">

Gift card campaigns:
- ID
- Type
- Source ID
- Reason
- Balance
- Amount
- Creation Date
- Voucher ID
- Campaign ID
- Source
- Details

Loyalty campaigns:
- ID
- Type
- Source ID
- Reason
- Balance
- Amount
- Creation Date
- Voucher ID
- Campaign ID
- Details
- Related Transaction ID

</Accordion>

### Export vouchers

Export voucher data to review or analyze voucher usage.

Follow these steps:

1. Go to **Vouchers**
2. Apply filters (optional)
3. Select **Manage filtered vouchers**
4. Select **Export filtered vouchers**

<Accordion title="Available voucher fields">

- ID
- Code
- Voucher Type
- Value
- Campaign
- Category
- Dates (creation, expiration)
- Balance fields
- Redemption data
- Customer data
- Discount settings
- Metadata

</Accordion>

### Export redemptions

You can export redemptions from the main Redemptions tab or from a selected campaign.

You can export:

- all redemptions
- filtered redemptions
- campaign-specific redemptions

<Accordion title="Available redemption fields">

- ID
- Date
- Voucher code
- Campaign
- Customer
- Order data
- Result
- Failure details
- Metadata

</Accordion>

After you confirm the export, Voucherify prepares the file.

Download it from **Notifications** when it is ready.

## Notes about exports

Exports are created first and downloaded later.

Date-time fields are exported in the **UTC timezone**.
